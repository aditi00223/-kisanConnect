# 🌾 KisanConnect — Seedha Kisan, Seedha Daam

A direct farmer-to-buyer agricultural marketplace that eliminates middlemen and ensures farmers get full price for their crops.

---

## 🚀 Features

- 🧑‍🌾 **Farmer Dashboard** — List crops, manage orders, track earnings
- 🛒 **Buyer Marketplace** — Browse & compare crop prices from multiple farmers
- 📦 **Order Placement** — Place orders with UPI or Cash on Pickup
- 📈 **Demand Forecasting** — View crop price trends over 6 months
- 👤 **Farmer Profile** — Stats, farm details, recent activity
- 🌍 **Multilingual** — English, Hindi, Punjabi support via i18next
- 📱 **Mobile First** — Responsive design for farmers on phones

## ⚙️ Setup Instructions

```bash
# Clone the repo
git clone https://github.com/aditi00223/-kisanConnect.git

# Go into frontend folder
cd -kisanConnect/frontend

# Install dependencies
npm install --legacy-peer-deps

# Start the app
npm start
```

App runs at **http://localhost:3000**
